## 1. コンセプト概要

| 項目 | 内容 |
|---|---|
| ジャンル | 2D重力反転パズルプラットフォーマー |
| テーマ | グリム童話の世界に取り込まれた主人公が本から脱出する |
| 目的意識 | 「物語を読み進める」＝「ページ＝ステージをクリアする」＝「脱出に近づく」 |
| コアギミック | 白紙の本を使い、今いるページの重力を反転させる |
| フォーマット | 1ステージ＝本の1見開き（左ページ＋右ページ） |
| 進行の証 | ステージクリアで主人公の白紙の本にそのページの内容（挿絵・文章）が記録されていく |

**世界観の軸**：主人公は物語の「読者」でも「登場人物」でもなく、本に迷い込んだ**異物**。白紙の本は「まだ書かれていない主人公自身の物語」であり、それをめくる（＝重力を反転させる）ことで、既に書かれた童話の世界のルールに干渉できる、という位置づけにすると、ゲームプレイとテーマが一致します。

## 2. コア構造：見開きページとステージ

### 2.1 基本ユニット
- 1ステージ＝1見開き（左ページ／右ページ）
- 中央に「ノド」（本の綴じ目）があり、キャラクターはここを自由に横断できる
- **オブジェクト（足場、敵、アイテム、ギミック）はそれぞれ所属するページ（左 or 右）に固定され、ノドを越えて移動しない**
- ゴール：見開きの中にある「ページをめくる」オブジェクト（本の角や栞など）に到達するとステージクリア→次の見開きへ

### 2.2 重力反転の適用範囲（要検討ポイント）
ここがゲーム設計の核となる部分なので、2案を比較します。

**案A：見開き全体（左右同時）に重力がかかる**
- シンプルで理解しやすい。左右で足並みを揃えた「対称パズル」が作りやすい
- ただし「左右非対称」の面白さが出しにくく、後半で飽きやすい

**案B：主人公が今いる側のページ（左 or 右）だけ重力が反転する（推奨）**
- 「左は通常重力、右は反転重力」という**非対称状態**を意図的に作れる
- 例：右ページで反転→天井にあった足場が床になる→そこを登ってノドへ→左ページ（通常重力）に戻る→再び右へ行くと右は反転したまま
- 「行ったり来たりしながら状態を組み合わせる」というパズルの奥行きが生まれ、後半ステージの難易度設計がしやすい
- 難点：プレイヤーが「今どちらのページがどの重力状態か」を把握できるUIが必須（後述）

→ **本仕様書では案Bを基本方針として採用**し、チュートリアル序盤のみ案A的な「対称パズル」で導入する設計を提案します。

### 2.3 重力反転のルール詳細
- ボタン一つで「本をひっくり返す」アクション（例：Xボタン／スペースキー）
- 反転対象は**主人公が現在立っている（または最後に接地していた）側のページのみ**
- 反転中、そのページに属する床・足場・敵・アイテムはすべて上下反転した重力に従う
- ノドを横断した瞬間、キャラクター自身の重力は「移った先のページの重力状態」に切り替わる（＝キャラの重力は本の状態ではなく、今いる場所の状態に追従）
- 反転にはクールダウンや回数制限を設けない方向を基本とするが、ステージによっては「反転回数制限」「反転できないゾーン（呪いの鎖など）」を後半の変化球として導入

## 3. プレイヤーキャラクター

| アクション | 内容 |
|---|---|
| 左右移動 | 通常のプラットフォーマー操作 |
| ジャンプ | 現在の重力方向に応じて「上」または「下」に跳ぶ |
| 本をめくる（重力反転） | 現在いるページの重力を反転 |
| ノド越え | 左右ページ間を自由に移動（オブジェクトは越えられない） |
| （将来案）持つ/投げる | 小物を持ってノドを越え、反対ページに置く→左右の制約を部分的に破る特殊ギミック |

## 4. UI／状態の可視化（重要）

案Bを採用する場合、プレイヤーが混乱しないための視覚設計が必須です。

- 画面下部または本のフチに「左ページ／右ページ」の重力状態を示す小さいアイコン（↑↓）を常時表示
- 反転していないページ＝インクが正常な向き、反転したページ＝インクが滲んで上下逆さの挿絵になる、など**アートと状態表示を一致させる**（絵本の紙・インクの質感と相性が良い）
- ノドを跨ぐ瞬間に軽い画面演出（紙がめくれる効果）を入れ、「今切り替わった」ことを体感的に伝える

## 5. 進行構造（本全体の設計）

```
本（ゲーム全体）
├─ 序章：本に取り込まれる／白紙の本を入手（チュートリアル）
├─ 第1章：赤ずきん（3〜5見開き＋章末ギミックページ）
├─ 第2章：ヘンゼルとグレーテル
├─ 第3章：シンデレラ
├─ 第4章：白雪姫
├─ 第5章：いばらひめ（眠れる森の美女）
└─ 終章：脱出（全ギミックを組み合わせた最終ページ群）
```

- 各章クリア時、白紙の本にその章の挿絵・一文が書き込まれる演出（＝主人公が自分の物語を取り戻していく比喩）
- 全章クリア後、白紙の本が「主人公自身の物語」として完成し、それが脱出の鍵になる、という終盤の伏線にできる

### 5.1 各章の「重力以外のプラスワン要素」
同じ重力反転だけだと単調になるため、各章に童話由来の固有ギミックを1つ足します。

| 章 | 固有ギミック案 | 重力反転との組み合わせ方 |
|---|---|---|
| 赤ずきん | 森の狼＝追跡者。音を立てると近づく | 反転で狼の頭上の木を落として妨害 |
| ヘンゼルとグレーテル | パンくず＝一時的な足場。時間で消える | 反転して落ちてくるパンくずを新たな床にする |
| シンデレラ | 「12時の鐘」タイマー。鳴ると変化した足場が元に戻る | 反転状態がリセットされる前に解く時間制パズル |
| 白雪姫 | 鏡ページ：右ページの動きが左ページに鏡写しで反映 | 左右非対称重力と鏡写しの組み合わせで多層パズル |
| いばら姫 | 棘が徐々に成長し通路を塞ぐ | 反転で棘の伸びる方向を変え、道を作る |

## 6. サンプルステージ設計（イメージ）

### 例1：チュートリアル（序章）
```
左ページ                | 右ページ
────────────────────────┼────────────────────────
床の上に主人公スタート   | 天井に足場、下は穴
そのまま右へ進める       | 反転しないと渡れない
                        | → ここで初めて「今いるページだけ反転」を学ばせる
```

### 例2：第2章 中盤（左右非対称パズル）
```
左ページ（通常重力）      | 右ページ（反転済み）
────────────────────────┼────────────────────────
鍵が天井付近の届かない位置 | ゴールの扉（下向き＝天井にある）
→ 右で反転して鍵を取り、  |
  左に戻り再度反転して    |
  通常に戻さないと鍵の位置に届かない、等の手順パズル |
```

このように「順番」「往復」「状態の記憶」を要求する設計が案Bの強みです。

## 7. アートディレクション（将来のプロトタイプ向けメモ）

ご希望の「絵本っぽい紙・インクの質感」を活かす方向性：

- 背景：古い紙のテクスチャ、経年劣化のシミやヨレ
- キャラクター・オブジェクト：木版画／銅版画風の線画＋水彩の滲み
- 重力反転時の演出：ページがインクごと上下逆さに「滲んで」反転するようなシェーダー表現
- 章ごとに使用するインクの色調を変える（赤ずきん＝赤茶系、白雪姫＝藍色系など）で視覚的に章の切り替わりを伝える
- 白紙の本が埋まっていく演出は、ペンで書き込まれるようなアニメーションにすると「物語を取り戻す」テーマと直結する
